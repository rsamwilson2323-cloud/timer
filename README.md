# ⏰ Countdown Timer with Alarm

An interactive HTML5 Countdown Timer application that allows users to set hours, minutes, and seconds with an automatic alarm notification when the timer reaches zero. Designed with a modern responsive interface, the application works seamlessly across desktop and mobile devices.

## ✨ Features

* ⏰ Custom countdown timer
* 🔔 Automatic alarm notification
* ▶️ Start timer functionality
* ⏸️ Pause timer support
* 🔄 Reset timer instantly
* 📱 Mobile-friendly design
* 🖥️ Responsive user interface
* ⚡ Real-time countdown updates
* 🎨 Modern glass-style layout
* 🚀 Lightweight and dependency-free

## 🎮 How to Use

1. Enter Hours, Minutes, and Seconds.
2. Click the Start button.
3. Watch the timer count down in real time.
4. Use Pause to temporarily stop the countdown.
5. Use Reset to clear the timer.
6. When the timer reaches zero, the alarm sound will play automatically.
